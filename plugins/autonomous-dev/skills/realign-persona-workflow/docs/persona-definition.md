# Stage 2: Persona Definition

Define clear character profile with measurable traits.

## Overview
Create comprehensive persona specification with core traits, style, and behavioral patterns.

## Core Traits (5-10)
Select key personality characteristics:
- **Friendly**: Warm, approachable, positive
- **Concise**: Clear, direct, efficient
- **Knowledgeable**: Demonstrates expertise
- **Patient**: Thorough explanations
- **Professional**: Respectful, appropriate

## Communication Style
- **Tone**: Conversational, formal, casual
- **Formality**: High, medium, low
- **Humor**: Level and type
- **Empathy**: Acknowledgment of user feelings

## Behavioral Patterns
- Greeting style
- Response structure
- Clarification approach
- Conclusion format

## Quality Gate
- 5-10 traits clearly defined
- Traits measurable
- Examples documented

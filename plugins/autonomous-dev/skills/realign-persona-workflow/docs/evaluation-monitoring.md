# Stage 7: Evaluation & Monitoring

Validate persona consistency effectiveness.

## Process
1. Measure final consistency score
2. Evaluate trait adherence per trait
3. Check style variance
4. Validate capability retention ≥95%
5. Compare to baseline

## Success Criteria
- ✅ Consistency score ≥85%
- ✅ Trait adherence ≥90%
- ✅ Style variance <15%
- ✅ Capability retention ≥95%

**See**: `capability-assessment.md` for regression detection.

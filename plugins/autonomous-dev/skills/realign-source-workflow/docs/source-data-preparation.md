# Stage 2: Source Data Preparation

Prepare source-grounded data with proper citations and indexing.

## Overview
Create comprehensive source database and claim-source pairs.

## Process
1. **Index Sources**: Build searchable database (Wikipedia, papers, documentation)
2. **Create Pairs**: Match claims to supporting sources
3. **Validate Retrieval**: Ensure precision ≥85%
4. **Generate Citations**: Format proper attributions

## Source Types
- Academic papers
- Official documentation
- Verified knowledge bases
- Expert-validated content

## Quality Gate
- Source retrieval precision ≥85%
- Diverse source types
- Clear attribution format

**See**: `../templates.md` for source data format.

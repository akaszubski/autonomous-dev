# Stage 7: Evaluation & Monitoring

Validate source attribution effectiveness.

## Process
1. Measure final citation accuracy
2. Evaluate attribution coverage
3. Check retrieval precision
4. Validate capability retention ≥95%
5. Compare to baseline

## Success Criteria
- ✅ Citation accuracy ≥90%
- ✅ Attribution coverage ≥80%
- ✅ Retrieval precision ≥85%
- ✅ Capability retention ≥95%

**See**: `capability-assessment.md` for regression detection.

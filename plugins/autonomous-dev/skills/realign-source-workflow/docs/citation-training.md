# Stage 3: Citation Training

Train model to properly cite sources and attribute information.

## Overview
Fine-tune model on citation examples to learn proper attribution.

## Process
1. Prepare citation training data
2. Fine-tune on attribution patterns
3. Test citation accuracy
4. Validate accuracy ≥90%

## Citation Patterns
- Inline citations: "According to [source], ..."
- Footnote style: "Claim.[1]"
- Bibliography: "Source: Author, Title, Year"

## Quality Gate
- Citation accuracy ≥90%
- Consistent format
- Proper attribution
